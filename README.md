# End-to-End Supervised Machine Learning System  
Cristian Negrin  
Midterm Mini-Project (Modules 1–7)

## Project Overview

This project implements a complete supervised machine learning system using mathematically grounded techniques including:

- Linear Algebra (matrix representation, rank, PCA)
- Optimization (Gradient Descent from scratch)
- Probability & Bayesian reasoning (Naive Bayes)
- Regression (Linear, Ridge, Lasso)
- Classification (Logistic Regression, KNN)
- SVM with kernels (Linear & RBF)
- Tree-based and Ensemble methods (Decision Tree, Random Forest, Gradient Boosting)

## Dataset

Energy Operations dataset  
Targets:
- Regression: `energy_kwh`
- Classification: `high_peak`

## Final Recommendation

Logistic Regression was selected as the final classification model due to strong F1 performance and interpretability.

For regression, Ridge provides stable performance under multicolarity.
